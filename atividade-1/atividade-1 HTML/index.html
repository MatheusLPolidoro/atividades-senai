Versionamento de software

UC7 - Versionamento

  1. Baixar o GIT no site official

  2. Instale o GIT em sua máquina

  3. Crie seu usuário e forneça um endereço de e-mail

  4. Crie um repositório GIT local

  5. Crie um arquivo para ser alterado e rastreado

  6. Faça seu cadastro no GitHub

  7. Crie um repositório on-line

  8. Publique seu trabalho e faça alterações

  9. Crie um README.md com os principais comandos utilizados
